## Description
MaTools is a multifunctional Python tool packed and compressed to protect the source code.  
It provides:

- IP information lookup  
- Internet speed test  
- Hardware statistics monitoring  
- Domain-to-IP resolution  

> ⚠️ Note: The code is packed using `marshal`, `zlib`, and `base64` to prevent direct copying or unauthorized modifications.
