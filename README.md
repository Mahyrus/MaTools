# MaTools v2.8

**Developer:** Mahyrus  
**Version:** 2.8

---

## Description
**MaTools** is a multifunctional Python tool that provides:

- **IP Information:** Local and Public IPv4, region detection  
- **Domain to IP:** Convert a domain name to its IP address  
- **Speed Test:** Test internet download/upload speed and ping  
- **Hardware Stats:** CPU name, cores, usage and RAM total & usage  

The script is **packed and compressed** to protect the source code using `marshal`, `zlib`, and `base64`.

> ⚠️ Packed code is harder to copy or modify directly.

---

## Requirements
- Python 3.x  
- Windows (for `msvcrt` support)  

Python modules (will be installed via `setup.bat`):

- `colorama`  
- `requests`  
- `psutil`  
- `speedtest-cli`  

Other modules used are **built-in Python modules**: `os`, `sys`, `time`, `json`, `socket`, `platform`, `threading`, `itertools`, `marshal`, `zlib`, `base64`.
